# clj-hexapod

This a control library for the
[PhantomX Hexapod](http://www.trossenrobotics.com/phantomx-ax-hexapod.aspx).

You can control it via XBee and
[UartSBEE](http://www.trossenrobotics.com/uartsbee) connected to your
computers USB port.
